Image Processing

Filenames

All images are assigned a UUID. This uuid serves as the filename for the file in all locations. If you want to find the same file in a different format or resolution, search for this uuid.

Original Images

Original images are stored in the /original/ directory. Images will have their uuid as the filename, and the original file extension.

Image Conversion

Images are converted into the following formats:

  • JPEG - Stored in /jpg/
  • PNG - Stored in /png/
  • WEBP - Stored in /webp/

Size Variants

Within each format folder, different resolution options are available:

  • 200: Images reduced to 200 pixels in width
  • 500: Images reduced to 500 pixels in width
  • 1000: Images reduced to 1000 pixels in width
  • Original: Images in their original size
