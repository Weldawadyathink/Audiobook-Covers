Overview

The Audiobook Covers API provides a simple interface to search for audiobook covers based on the text in the cover image. It has one primary endpoint:

  • GET /cover/bytext/

API Endpoint

GET /cover/bytext/

This endpoint retrieves a list of audiobook covers that match the specified search text.

Request Parameters

Parameter Type Description
q string Search query based on the text in the cover image. (required)

Response

The response is a JSON array of objects, each containing the following properties:

Property Type Description
filename string URL of the audiobook cover image.
source string URL for the source of the image.

Example Request

GET https://api.audiobookcovers.com/cover/bytext/?q=harry%20potter
